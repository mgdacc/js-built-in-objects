¿Qué son los Objetos Built-in?

Son objetos que JavaScript proporciona de forma nativa en su entorno de ejecución, ofreciendo funcionalidades esenciales sin necesidad de instalar librerías externas.

// No necesitas importar nada para usar 'Math.random()'
// o para crear una nueva fecha con 'new Date()'.
// ¡Ya vienen incluidos!
2 / 20

String

Proporciona propiedades y métodos para trabajar con cadenas de texto. Cada string que creas tiene acceso a estas herramientas.

let mensaje = "JavaScript es Genial";

// Propiedad para obtener la longitud
console.log(mensaje.length); // 18

// Métodos para cambiar mayúsculas/minúsculas
console.log(mensaje.toUpperCase()); // "JAVASCRIPT ES GENIAL"
console.log(mensaje.toLowerCase()); // "javascript es genial"
3 / 20

Métodos Útiles de String

Permiten buscar, extraer y reemplazar partes de un texto.

let frase = "Hola Mundo, bienvenido al Mundo JS";

// Encontrar la posición de una subcadena
frase.indexOf('Mundo'); // 5

// Extraer una parte del string
frase.slice(5, 10); // "Mundo"

// Reemplazar texto
frase.replace('Mundo', 'Universo'); // "Hola Universo, bienvenido al Mundo JS"

Number

Ofrece métodos para formatear y analizar números.

let precio = 49.9567;

// Formatear a un número de decimales fijos
precio.toFixed(2); // "49.96" (devuelve un string)

// Convertir un string a un número entero
Number.parseInt("101.5 manzanas"); // 101

// Comprobar si un valor NO es un número
Number.isNaN('hola'); // true

Boolean y Function

Son los constructores para los primitivos `boolean` y las `function`. Aunque se usan menos directamente, son fundamentales para el lenguaje.

let esVerdadero = true;
// esVerdadero.toString() -> "true"

function saludar() {}
// saludar.name -> "saludar"
// saludar.length -> 0 (número de argumentos esperados)

Math (Objeto Estático)

Es un objeto singular (no se instancia con `new`) que agrupa constantes y funciones matemáticas.

// Constantes matemáticas
console.log(Math.PI); // 3.141592653589793
console.log(Math.E);  // 2.718281828459045

// Generar un número aleatorio entre 0 y 1
console.log(Math.random());

Math: Redondeo

Funciones para redondear números de diferentes maneras.

let numero = 9.5;

Math.floor(numero); // 9 (redondea hacia abajo)
Math.ceil(numero);  // 10 (redondea hacia arriba)
Math.round(numero); // 10 (redondeo estándar al más cercano)
8 / 20

Math: Potencias y Raíces

Cálculos comunes como potencias, raíces, máximos y mínimos.

// Potencia (2 elevado a la 8)
Math.pow(2, 8); // 256

// Raíz cuadrada
Math.sqrt(81); // 9

// Encontrar el valor máximo o mínimo
Math.max(10, 5, 100, 2);  // 100
Math.min(10, 5, 100, 2);  // 2

Date: Creación de Fechas

El objeto `Date` se usa para trabajar con fechas y horas. Se debe instanciar con la palabra clave `new`.

// Fecha y hora actual
const ahora = new Date();
console.log(ahora);

// Una fecha específica (Año, Mes-1, Día)
const navidad = new Date(2025, 11, 25);
console.log(navidad);

Date: Obtener Componentes

Una vez creado un objeto `Date`, podemos extraer sus partes individuales.

const hoy = new Date();

hoy.getFullYear(); // Devuelve el año (ej: 2025)
hoy.getMonth();    // Devuelve el mes (0-11, Enero es 0)
hoy.getDate();     // Devuelve el día del mes (1-31)
hoy.getHours();    // Devuelve la hora (0-23)
hoy.getDay();      // Devuelve el día de la semana (0-6, Domingo es 0)

Date: Formateo

Existen varios métodos para convertir una fecha en un string legible para humanos.

const evento = new Date();

// Formato de fecha estándar
evento.toDateString(); // "Fri Nov 14 2025"

// Formato de hora local
evento.toLocaleTimeString(); // "11:54:00 PM"

// Formato estándar ISO 8601 (muy usado en APIs)
evento.toISOString(); // "2025-11-15T05:54:00.000Z"

Array: Métodos básicos

Probablemente el objeto más usado. Ofrece una enorme cantidad de métodos para manipular listas de datos.

let frutas = ["Manzana", "Banana"];

// Añadir al final
frutas.push("Naranja"); // ["Manzana", "Banana", "Naranja"]

// Eliminar el último
frutas.pop(); // ["Manzana", "Banana"]

// Unir elementos en un string
frutas.join(", "); // "Manzana, Banana"
13 / 20

Array: Métodos Iterativos

Métodos modernos y muy potentes que reciben una función como argumento para procesar los elementos del array.

let numeros = [1, 2, 3, 4, 5];

// .map(): Crea un nuevo array con los resultados
const dobles = numeros.map(n => n * 2); // [2, 4, 6, 8, 10]

// .filter(): Crea un nuevo array con elementos que cumplen una condición
const pares = numeros.filter(n => n % 2 === 0); // [2, 4]

Object

El ancestro de casi todos los objetos en JS. Proporciona métodos estáticos para trabajar con las claves y valores de cualquier objeto.

const coche = { marca: 'Tesla', modelo: 'Model S' };

// Obtener un array con las claves
Object.keys(coche);   // ["marca", "modelo"]

// Obtener un array con los valores
Object.values(coche); // ["Tesla", "Model S"]

// Obtener un array de arrays [clave, valor]
Object.entries(coche); // [["marca", "Tesla"], ["modelo", "Model S"]]

Error

Es el objeto base para los errores. Se utiliza junto con las sentencias `throw` y `try...catch` para manejar excepciones en el código.

try {
  // Código que podría fallar...
  throw new Error("Algo salió muy mal");
} catch (error) {
  // Capturamos el objeto Error
  console.error(error.message); // "Algo salió muy mal"
}

JSON (Objeto Estático)

Proporciona métodos para convertir objetos de JavaScript a formato de texto JSON (JavaScript Object Notation) y viceversa.

const datos = { id: 42, user: "admin", active: true };

// Convertir un objeto a un string JSON
const jsonString = JSON.stringify(datos);
// '{"id":42,"user":"admin","active":true}'
17 / 20

JSON.parse()

Es la operación inversa: convierte una cadena de texto con formato JSON en un objeto de JavaScript con el que podemos trabajar.

const jsonString = '{"id":101,"producto":"Laptop"}';

// Convertir un string JSON a un objeto
const productoObj = JSON.parse(jsonString);

console.log(productoObj.producto); // "Laptop"
